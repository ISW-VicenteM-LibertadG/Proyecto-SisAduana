# Proyecto SisAduana

## Descripción general

SisAduana es un prototipo web desarrollado para la asignatura **Ingeniería de Software** de **DUOC UC**, cuyo objetivo es apoyar los procesos de control fronterizo mediante una plataforma digital que facilite el registro de pasajeros, la gestión de usuarios, la validación documental y la generación de reportes estadísticos.

El proyecto fue diseñado utilizando **Figma**, lo que permitió simular el funcionamiento del sistema mediante un prototipo navegable sin necesidad de implementar código.

## Entregables del repositorio

El repositorio se encuentra organizado de la siguiente manera:

```text
prototipo/
│── Prototipo web SisAduana.make
│── README.md

pruebas/
│── Documento de pruebas prototipo SisAduana.pdf

versionado/
│── cambios.md

coevaluacion/
│── Informe de Coevaluación.pdf

README.md
```

### Contenido de cada carpeta

- **prototipo/**: Contiene los archivos exportados del prototipo desarrollado en Figma junto con las instrucciones de navegación del sistema.
- **pruebas/**: Incluye el Plan de Pruebas basado en el estándar ISO/IEC 25000 y la planilla de casos de prueba utilizados para validar el prototipo.
- **versionado/**: Contiene el registro de iteraciones del proyecto y la justificación de los cambios realizados durante su desarrollo.
- **coevaluacion/**: Incluye el informe de coevaluación realizado al grupo asignado.
- **README.md**: Documento con la descripción general del proyecto, integrantes y estructura de los entregables.

Junto con el acceso a este repositorio GitHub, la entrega adjuntará el archivo con el informe completo y el enlace al prototipo.

## Funcionalidades principales

El prototipo implementa las siguientes funcionalidades:

- Inicio de sesión con autenticación por roles.
- Gestión de usuarios.
- Registro de ingreso de pasajeros.
- Validación documental para menores de edad.
- Gestión de ingreso temporal de vehículos.
- Implementación de la Declaración Jurada SAG.
- Generación de reportes estadísticos.
- Registro de actividad mediante logs del sistema.

## Objetivo del proyecto

Desarrollar un prototipo funcional de un sistema de apoyo para el Servicio Nacional de Aduanas que permita optimizar los procesos de control fronterizo mediante una interfaz intuitiva, segura y alineada con los requisitos funcionales y no funcionales definidos para el proyecto.
